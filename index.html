margin or padding

  • 0 - sets margin or padding to 0
  • 1 - sets margin or padding to .25rem (4px if font-size is 16px)
  • 2 - sets margin or padding to .5rem (8px if font-size is 16px)
  • 3 - sets margin or padding to 1rem (16px if font-size is 16px)
  • 4 - sets margin or padding to 1.5rem (24px if font-size is 16px)
  • 5 - sets margin or padding to 3rem (48px if font-size is 16px)
  • x - sets both left and right
  • y - sets both top and bottom

    * Flex

    uses flexbox, instead of floats, to handle the layout.

    Flex item 1
    Flex item 2
    Flex item 3

    外層div.flex-container屬性：

    display: flex; 裡面包覆的div寬度(沒有設定)，隨內容彈性長，但內文會跑出去，可以增加word-break: break-all;高度就會同時被撐高

    裡面div超出都會跑出

    • flex-direction: column(直排); column-reverse(直排＋反轉); row-reverse(橫排＋反轉);
    • flex-wrap: wrap; nowrap; wrap-reverse;
    • flex-flow: row wrap; (合併 flex-direction , flex-wrap)
    • justify-content: center; space-between; ... (置中， 左右中各自對齊，)
    • align-items:center 靠左水平置中 ; flex-start 靠左水平齊上; flex-end 靠左水平齊下; stretch (default); baseline (對齊文字底線);
      (在flex-container裡面的div對齊方式: )
    • align-content: space-between 上下撲滿 空白中間平衡; space-around 空白上中下平衡; stretch div撲滿高度(default); center 左邊中間開始對齊; flex-start 左上開始排列; flex-end; 左下開始排列
      (裡面div排列方式)

    外層div.flex-container 範例：畫面內 div水平垂直都置中

    使用
    • justify-content: center;
    • align-items: center;

    內層子div

  • order 順序
  • flex-grow 滿版相對水平比例
  • flex-shrink 收縮0~2
  • flex-basis 指定寬度
  • flex (flex-grow, flex-shrink, and flex-basis)
    範例：flex:0 0 30%;
  • align-self 內容對齊，影響高度


    • 在row內會受flex:1;影響佈滿高度，需在col裡面
